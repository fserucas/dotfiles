🎹 KANATA ENHANCED KEYBOARD LAYOUT REFERENCE
==================================================

OVERVIEW
--------
This layout optimizes all three keyboard rows for maximum efficiency:
- Top Row: Programming symbols and function keys
- Home Row: Modal modifiers (Meta, Alt, Shift, Ctrl)
- Bottom Row: Text editing shortcuts
- 4 Specialized Layers: Navigation, Numbers, Symbols, Editing

LAYER ACCESS SUMMARY
=====================
🧭 Navigation Layer: Hold SPACE
🔢 Number Layer:     Hold TAB

QUICK TIPS
==========
• Home row modifiers work on all layers
• Hold timing: 200ms (configurable)
• All layers stack with base modifiers
• Bottom row shortcuts work without layers
• Top row brackets accessible directly

File: /tmp/dotfiles/home/.kanata.conf
